# Air Quality Data Analysis Dashboard

Dashboard interaktif ini dibuat menggunakan **Streamlit** untuk memvisualisasikan data kualitas udara (khususnya polutan PM2.5 dan hubungannya dengan cuaca) dari berbagai stasiun pengamatan di Beijing.

## Prasyarat
Pastikan Anda memiliki hal-hal berikut terinstal di komputer Anda:
- Python 3.9 atau lebih baru
- Terminal atau Command Prompt

## Cara Menjalankan Dashboard di Lokal

Ikuti langkah-langkah berurutan di bawah ini untuk menjalankan aplikasi di komputer Anda (contoh menggunakan MacOS/Linux terminal):

**1. Clone atau Unduh Repository**
Pastikan Anda sudah berada di dalam folder proyek (tempat file `README.md` ini berada).
